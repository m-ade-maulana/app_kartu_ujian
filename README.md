# SIKU (Sistem Informasi Kartu Ujian)

## Apa itu SIKU ?

SIKU adalah sistem untuk melihat dan mencetak kartu ujian untuk seluruh peserta ujian.
seluruh informasi mengenai ujian yang akan berlangsung dapat dilihat oleh seluruh siswa.
SIKU dikembangkan untuk dapat memudahkan pekerjaan bagian tata usaha sekolah dalam menangani urusan manajemen terkait kegiatan ujian peserta didik.

## Fitur pada aplikasi

Sistem ini memiliki 3 hak akses yaitu admin, pengawas dan peserta didik.

Admin

- Daftar Peserta (edit ruangan ujian dan akses kartu ujian)
- Cetak kartu (Fitur untuk mencetak kartu ujian)
- Data pengawas (menambahkan, mengubah dan menghapus pengawas dari daftar)
- Jadwal ujian (Menambahkan mata pelajaran yang di ujikan)

Pengawas

- Daftar peserta (Melihat daftar peserta yang berada di ruangan)

Peserta didik

- Cetak kartu (Mencetak kartu dalam bentuk PDF)
